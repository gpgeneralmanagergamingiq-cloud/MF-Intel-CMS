│                                                                       │
│  🔧 TROUBLESHOOTING:                                                 │
│                                                                       │
│  ❌ Bridge won't start?                                              │
│     → Check Node.js is installed: node --version                     │
│     → Reinstall dependencies: cd nfc-bridge && npm install           │
│                                                                       │
│  ❌ Reader not detected?                                             │
│     → Check Device Manager → Smart card readers → ACS ACR122 0       │
│     → Unplug and replug USB cable                                    │
│     → Restart bridge service                                         │
│                                                                       │
│  ❌ Can't connect from browser?                                      │
│     → Make sure bridge is running (shows "Ready to scan")            │
│     → Check no firewall blocking localhost:8765                      │
│     → Refresh browser page                                           │
│                                                                       │
│  ❌ Card not scanning?                                               │
│     → Try different card (some need to be held closer)               │
│     → Check blue LED on ACR122U lights up                            │
│     → Wait 1-2 seconds for detection                                 │
