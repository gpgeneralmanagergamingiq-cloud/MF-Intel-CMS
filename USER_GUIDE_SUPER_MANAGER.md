### Financial Overview Card

**[SCREENSHOT: Financial metrics card]**

```
┌─────────────────────────────────────────┐
│  💰 FINANCIAL OVERVIEW                  │
├─────────────────────────────────────────┤
│  Total Drop: CFA 45,000,000             │
│  Total Win: +CFA 2,250,000              │
│  Hold %: 5.00%                          │
│                                         │
│  [View Drop by Table] →                 │
└─────────────────────────────────────────┘
```

**Understanding the Numbers:**

**Drop (CFA 45,000,000)**
- Total cash inserted into table drop boxes today
- This is your "handle" or total cash wagered
- Higher drop = more player activity

**Win (+CFA 2,250,000)**
- Casino profit for the day
- Positive = Casino won
- Negative = Casino lost (bad day)

**Hold % (5.00%)**
- Win divided by Drop: (2,250,000 ÷ 45,000,000) × 100
- Industry standard: 4-8%
- **5% is healthy!** ✅

**How to Use This:**
- Monitor hold percentage daily
- Investigate if hold % drops below 3% (potential issues)
- Celebrate when hold % is healthy
- Track trends over time

### When to Use Each Type

**Super Manager** ⭐⭐⭐
- **WHO:** Casino owner, General Manager
- **WHEN:** Need full system control
- **COUNT:** 1-2 users (keep limited!)

**Manager** ⭐⭐
- **WHO:** Shift Manager, Floor Manager
- **WHEN:** Need operational control, approvals
- **COUNT:** 3-5 users (one per shift)

**Pit Boss** ⭐
- **WHO:** Pit Manager, Floor Supervisor
- **WHEN:** Manage all table operations
- **COUNT:** 2-4 users (based on floor size)

**Inspector**
- **WHO:** Table supervisor, game supervisor
- **WHEN:** Monitor specific tables, rate players
- **COUNT:** 10-20 users (more than tables)

**Cashier**
- **WHO:** Cage staff, cash handlers
- **WHEN:** Cage operations only
- **COUNT:** 5-10 users (shift coverage)

**Host**
- **WHO:** Player services, VIP host
- **WHEN:** Player management, comps approval
- **COUNT:** 2-5 users (player volume dependent)

**Waiter**
- **WHO:** Floor servers, beverage staff
- **WHEN:** Record comps only
- **COUNT:** 10-20 users (floor coverage)

### Tier Criteria

**New** (First-time players)
- **Criteria:** First visit, no history
- **Benefits:** Welcome bonus, basic comps
- **Goal:** Convert to Regular

**Regular** (Standard members)
- **Criteria:** 3+ visits OR CFA 1M+ lifetime drop
- **Benefits:** Standard comp rate, basic promotions
- **Goal:** Identify potential VIPs

**VIP Silver** (Growing value)
- **Criteria:** 10+ visits OR CFA 5M+ lifetime drop
- **Benefits:** 10% higher comp rate, priority service
- **Goal:** Develop loyalty

**VIP Gold** (High value)
- **Criteria:** 25+ visits OR CFA 15M+ lifetime drop
- **Benefits:** 20% higher comp rate, special events
- **Goal:** Maintain and grow

**VIP Platinum** (Highest value)
- **Criteria:** 50+ visits OR CFA 50M+ lifetime drop
- **Benefits:** 30% higher comp rate, personal host, exclusive access
- **Goal:** Exceptional service, retention

# 7. FLOAT OPERATIONS OVERVIEW

As Super Manager, you **oversee** float operations but typically don't perform them directly (cashiers do). However, you need to understand and monitor.

## What You Need to Know

**Float = Table bankroll** (chips/cash at table)

**Operations:**
- **Open Float** - Start table with initial chips
- **Close Float** - End table, return chips
- **Fill** - Add chips mid-shift (running low)
- **Credit** - Remove chips mid-shift (excess)

**Your Role:**
- Monitor float activity
- Review discrepancies
- Approve large fills (optional policy)
- Audit end-of-day reconciliation

---
