### METHOD 3: Pandoc (PROFESSIONAL - 2 minutes)

**Perfect for advanced users and automation!**

**Step-by-Step:**

1. **Install Pandoc:**
   ```
   Windows: Download from https://pandoc.org/installing.html
   Mac: brew install pandoc
   Linux: sudo apt-get install pandoc
   ```

2. **Open Terminal/Command Prompt**

3. **Navigate to guide folder:**
   ```bash
   cd path/to/your/guides
   ```

4. **Convert ALL guides at once:**

   **Windows (Command Prompt):**
   ```cmd
   for %f in (USER_GUIDE_*.md) do pandoc "%f" -o "%~nf.pdf" --toc --toc-depth=2
   ```

   **Mac/Linux (Terminal):**
   ```bash
   for file in USER_GUIDE_*.md; do
     pandoc "$file" -o "${file%.md}.pdf" --toc --toc-depth=2
   done
   ```

5. **Result:**
   ```
   ✅ All 7 PDFs created in 1-2 minutes!
   ```

**Time: 2 minutes total for all 7** ⚡⚡⚡

**Quality: ⭐⭐⭐⭐⭐ PROFESSIONAL**

**Bonus:** Includes automatic table of contents!
